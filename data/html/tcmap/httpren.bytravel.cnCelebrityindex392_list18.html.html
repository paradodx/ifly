张兴祖 [ 明 ] 明朝开国功臣

张德胜 [ 明 ] 明朝开国功臣 ( 1331 ～ 1363 )

叶升 [ 明 ] 明朝开国功臣 (?～ 1392 )

赵庸 [ 明 ] 明朝开国功臣
