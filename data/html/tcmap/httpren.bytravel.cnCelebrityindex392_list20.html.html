李经邦 [ 清 ] ( 1852 ～ 1910 )

王尚辰 [ 清 ] 晚清“庐州三怪” ( 1826 ～ 1902 )

徐子苓 [ 清 ] 晚清“庐州三怪” ( 1812 ～ 1876 )
